## 十、数据库表结构

| 表名 | 说明 | 主要字段 |
|------|------|----------|
| user | 用户表 | id, username, password, phone, email, avatar, role, status, balance |
| game_account | 游戏账号表 | id, user_id, game_type, account_name, account_level |
| service | 服务表 | id, provider_id, game_type, service_type, title, price, duration, status, rating |
| orders | 订单表 | id, order_no, user_id, provider_id, service_id, total_amount, status |
| review | 评价表 | id, order_id, user_id, provider_id, rating, content, reply |
| message | 消息表 | id, sender_id, receiver_id, type, content, is_read |
| conversation | 会话表 | id, user1_id, user2_id, last_message, last_message_time |
| service_tag | 服务标签表 | id, name |
| service_tag_relation | 服务标签关联表 | id, service_id, tag_id |
| service_favorite | 服务收藏表 | id, user_id, service_id |
